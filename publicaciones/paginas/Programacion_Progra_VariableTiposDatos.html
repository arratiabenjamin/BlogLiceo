Variables y Tipos de Datos

04 de agosto, 2022

Descripción

Una variable es como una caja donde almacenar cosas solo que en una variable se guardan datos con un tipo definido, puedes almacenar palabras, frases, numeros enteros y decimales, valores verdaderos y falsos, lista de datos, etc.
En Pseint se debe definir que tipo de dato se almacenará en cada variable, pero hay lenguajes de programación como Python donde puedes darle un tipo de dato y luego ponerle un dato totalmente diferente, por ejemplo la variable "var1" contendra el valor de 1 y por un x caso debes cambiarle su dato a "hola", dato totalmente distinto a 1, Python te dejará sin ningún error pero en Pseint, JavaScript, Java, etc, no te dejará y te dará error.

Tipos de Datos

- Entero/Int: Este tipo de dato almacena datos que solo sean enteros si le pones decimal dara error.
- Decimal/Real/Float: Este igualmente almacena numeros, pero decimales también aceptará enteros pero le pondrá un .0
- Logico/Real/Bool: Tipo de dato que solo almacena dos datos Verdadero/True o Falso/False.
- Cadena/Carácter/String: Este te almacenará carácteres, Cadena de textos, numeros pero serán tomados como carácteres y no numeros.
